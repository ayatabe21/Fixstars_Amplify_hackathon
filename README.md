# amplify_hackathon

## 実行方法

1. ファイル *amplify_hackathon.py* とフォルダ *template_data* と *test_data* を同じディレクトリに置く

2. ファイル *amplify_hackathon.py* の、`client.token`のコメントアウトを外してアクセストークンを入力する

3. *amplify_hackathon.py* を実行する

```shell
$ python amplify_hackathon.py
```
